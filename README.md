# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**. The project provides a clean user interface and smooth gameplay for two players.

## 🚀 Live Demo

🔗 https://irshadansaridev.github.io/Tic-Tac-Toe-Game/

## ✨ Features

- Interactive two-player gameplay
- Responsive design
- Win and draw detection
- Restart game functionality
- Clean and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
